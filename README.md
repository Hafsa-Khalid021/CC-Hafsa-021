## Lab Tasks

1. **Download VMWare Workstation**  
   Install the virtualization software required for creating virtual machines.

2. **Download Ubuntu Server**  
   Obtain the Ubuntu Server ISO file.

3. **Browse the ISO file**  
   Select the downloaded ISO file in VMWare.

4. **Create a new virtual machine**  
   Set up a new VM on VMWare using the Ubuntu Server ISO.

5. **Choose your language**  
   Select the preferred language during installation.

6. **Choose the correct keyboard layout**  
   Configure the keyboard layout.

7. **Choose your install**  
   Select the appropriate installation type for Ubuntu Server.

8. **Networking**  
   Configure the network settings for the VM.

9. **Configure storage**  
   Allocate and set up storage for the VM.

10. **Confirm changes**  
    Review and confirm installation configurations.

11. **Set up a Profile**  
    Create a username and password for the server.

12. **Get the IP address**  
    Use the following command inside the VM:
    ```bash
    ip addr
    ```

13. **SSH Command on Windows**  
    Connect to the Ubuntu server from Windows using:
    ```bash
    ssh <username>@<ip-address>
    ```
